# Roadmap

## v1.1

- Persistir histórico de internet no SQLite.
- Alertas visuais para internet instável.
- Melhorar tratamento de encoding em dados antigos.
- Export de relatório Markdown.

## v1.2

- Autenticação local.
- Perfis de usuário somente leitura/admin.
- Configuração pela interface.
- Página dedicada de configurações.

## v1.3

- Docker e Docker Compose.
- Modo serviço no Windows.
- Deploy local em mini PC ou Raspberry Pi.

## v1.4

- Notificações locais.
- Webhook opcional para alertas defensivos.
- Regras customizáveis de risco.

## v1.5

- IA analisando eventos.
- Sugestões defensivas em linguagem natural.
- Resumos semanais de postura da rede.

## Ideias Futuras

- Comparação de snapshots.
- Modo demonstração com dados fictícios.
- Tema claro/escuro.
- Export PDF.
- Integração com Home Assistant.
