# Roadmap - Cyber Home Dashboard

Este roadmap registra direções de estudo e evolução do projeto. Itens listados aqui são planos, não funcionalidades já entregues.

## Próxima iteração

- Alertas para eventos importantes.
- Estatísticas mais claras sobre dispositivos e conectividade.
- Mapa visual da rede com melhor organização.
- Monitoramento em tempo real mais completo.
- Melhorias de UX no dashboard.
- Persistir histórico de conectividade no SQLite.
- Melhorar filtros temporais de eventos e métricas.
- Adicionar modo de demonstração com dados totalmente fictícios.
- Gerar relatório exportável para revisão local.

## Segurança e acesso

- Criar autenticação local para uso além de `127.0.0.1`.
- Definir perfis somente leitura e administrador.
- Tornar configurações sensíveis mais visíveis para revisão antes de scans.

## Distribuição

- Preparar Docker e Docker Compose.
- Investigar execução local contínua em Windows e Linux.
- Documentar deploy em mini PC ou Raspberry Pi.

## Automação e IA

- Notificações locais para eventos relevantes.
- Resumos defensivos de eventos com IA, sem automatizar ações invasivas.
- Recomendações explicáveis para inspeção manual de anomalias.

## Ideias futuras

- Integração com Home Assistant.
- Exportação em PDF.
- Comparação visual de snapshots da rede ao longo do tempo.
